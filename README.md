# UI Foundry Badge Component Generator

Standalone visual badge generator built with Next.js.

This app is a focused badge-building studio for status chips, labels, counters, notification markers, and branded mini-surfaces that can be edited visually and exported as code.

## Product summary

This generator is designed for:

- designers exploring status, label, and tag treatments quickly
- developers who want editable badge starting points for dashboards, admin panels, and product UI
- freelancers and agencies building notification and metadata surfaces for clients
- buyers who want a dedicated badge tool with the same UX language as the rest of UI Foundry

## What the product includes

- section-based editing controls
- live preview panel
- code view with copy support
- file export support
- searchable preset library
- generated premium preset catalog with filters, paging, and surprise-me apply
- undo and redo controls
- self-contained standalone app structure

## Editing coverage

The editor currently includes these control areas:

- Presets
- Content
- Appearance
- Status and Dot
- Effects
- 3D Engine
- A11y

These sections cover label content, icon and counter presentation, appearance and shape, status-dot behavior, effects and depth styling, and accessibility metadata.

## Export workflow

The badge preview and code output stay tied to the same current state so the exported result matches the visual editor closely.

## Why this product is useful

This is not a single prebuilt chip. It is an editable badge studio for status labels, counters, and UI markers that can be tuned to different product directions.
